About This Map

This interactive map combines the power of Google Maps API with OpenStreetMap data, styled with Material Design 3 expressive design principles. Features include:

  • Responsive design with MD3 color system and typography
  • Location search with geocoding
  • Current location detection
  • Zoom controls and fullscreen mode
  • Dark mode support
  • Accessible design patterns
